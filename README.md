# 🌾 2D Farming RPG & Adventure

A 2D Farming RPG game developed in Unity. This project was built not just as a game, but as a practical laboratory to apply advanced **Software Architecture**, **Object-Oriented Programming (OOP)**, and **Artificial Intelligence** concepts in C#.

## 🚀 Technical Highlights

This repository demonstrates the application of industry-standard programming practices to solve complex systemic challenges:

* **Design Patterns (Singleton):** Implemented the Singleton pattern (`DialogueControl`) to manage global dialogue states, ensuring a single point of access and preventing memory leaks across scenes.
* **Data-Driven Design (ScriptableObjects):** Architected a modular, multi-language dialogue system using `ScriptableObjects`. This decouples raw text data from the game logic, allowing for highly scalable localization and custom Unity Editor tooling.
* **Artificial Intelligence (NavMesh):** Engineered intelligent enemy behavior using Unity's NavMesh for dynamic pathfinding and obstacle avoidance. The AI calculates distances and makes decisions (patrol, chase, attack) based on player proximity.
* **Strict OOP & Encapsulation:** Applied robust object-oriented principles to manage the player's state machine (digging, watering, attacking, rolling). Utilized C# Properties (`get`/`set`) and access modifiers to securely encapsulate inventory limits, resource farming, and dynamic UI data binding.
* **Probability & State Machines:** Developed farming and fishing mechanics relying on transition states and RNG (Random Number Generation) algorithms to calculate drop rates and resource growth.
